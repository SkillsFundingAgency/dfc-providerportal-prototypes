Course file format requirements

Your data upload file must be a CSV file type. If you have more than one option when saving as a CSV, choose CSV UTF-8 (Comma delimited) (.csv).

The tables below show information about each field in the data upload file and how to complete it correctly.

  2. Who this course is for

  5. How you will learn

  8. Where next

LARS_QAN

Regulated and non regulated qualifications are listed by LARS (Learning Aims Reference) or QAN (Qualification Accreditation Number) code.

Codes can be found at Find a learning aim.

Mandatory
Yes
Character limit
8
Example
50079281
Z9483628
00280206
5011752X
Notes
Codes must be 8 digits. For codes with less than 8 digits add one or more zeros to the beginning of the code.

WHO_THIS_COURSE_IS_FOR

Information that will help the learner decide whether this course is suitable for them, the learning experience and opportunities they can expect from the course.

Mandatory
Yes
Character limit
2000
Notes
Avoid using special characters and line breaks.

ENTRY_REQUIREMENTS

START_DATE

Mandatory
When course has a specific start date
Example
24/09/2022
Notes

Start dates must be after today's date.

For courses with more than one start date, each course must be entered on a separate row.

If your course does not have a specific start date leave blank.
